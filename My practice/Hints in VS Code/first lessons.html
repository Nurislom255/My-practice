VS Code hints

Selecting multiple items at once

Using Alt + Click

Click on the first item, then hold Alt and click on the other items you want to select.

Column Selection (The "Vertical" Way)

Hold Ctrl + Alt and then press arrow buttons you want to change

  1. Place the cursor where you want to start the selection.
  2. Hold down Alt and Ctrl keys (Option key on Mac).
  3. Press the down/uparrow
  4. Release the mouse button, Ctrl key, and the Alt key when done.
  5. then write multiple lines of code at once!

Selection by match (the fastest way)

If you want to select every item "X" by match,

  1. Highligh the text you want to select
  2. Press Ctrl + D
  3. Then press again if you want to find the next match
