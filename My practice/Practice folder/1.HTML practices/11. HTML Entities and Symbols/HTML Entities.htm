HTML Entities

HTML Entities are used to describe text that are part of HTML symbols because these may be confused with HTML tags.

You can use these below for HTML entities:

  • &entity_name;
    • Or
  • &#entity_number;

</> is confused with HTML closing tag (this is just one example)


Thus, we use &lt; and &gt; instead of < and >

HTML symbols

HTML symbols are used to represent special characters which do not exist in your keyboard HTML.

They are represented by &symbol_name; or &#symbol_number;


Here are some common HTML symbols:

  • &reg; for Registred trademark symbol ®
  • &copy; for Copyright symbol ©
  • &trade; for Trademark symbol ™
for entity numbers:
  • &#174; for Registered trademark symbol ®
  • &#8364; for Euro symbol €
  • &#162; for Cent symbol ¢
