HTML Iframes Guide

An <iframe> (Inline Frame) is essentially a window that allows you to embed another HTML document (like a YouTube video, a Google Map, or another website) inside your current page.

1. Essential <iframe> Attributes

Attribute Purpose Why it's useful
src The URL of the page to embed. The most critical attribute.
title A description of the content. Vital for accessibility (screen readers).
loading="lazy" Delays loading until visible. Speeds up your initial page load.

2. Security: The sandbox Attribute

It restricts what the embedded content can do, protecting your site from malicious code.

<iframe src="https://example.com" 
        sandbox="allow-scripts allow-same-origin">
</iframe>
  • allow-scripts: Allows JavaScript.
  • allow-forms: Allows form submissions.
  • Empty sandbox: Blocks everything (highest security).

4. Common Pitfalls

X-Frame-Options: Many sites (Google, Facebook) block embedding to prevent "Clickjacking." If the frame is blank or says "Refused to connect," this is the reason.

HTTPS vs HTTP: You cannot embed an http site into an https site.
